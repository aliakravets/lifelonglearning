Front-end чи Back-end

Front-end

це та частина програмного забеспечення з якою користувач взаємодіє беспосередньо

Back-end

складова програмного забезпечення яка відповідає за логіку і функціонал

HTML i CSS - ключові іструменти для створення сайту,без них не написаний жоден сайт HTML - використовується для створення структури веб-сторінок,визначаючи елементи контенту,такі як текст,зображення і відео,та їх розташування CSS - відповідає за вигляд і форматування веб-сторінок,включаючи кольори, шрифти, відступи, рамки і інші стилізовані ефекти. Він дозволяє розробникам керуват виглядом контенту і забезпечує єдність дизайну на всьому сайті

JavaScript-Обовязковий інструмент Front-end розробнику

  • Зручність
  • Швидкість
  • Багатофункціональність

3 етапи розробки сайту

HTML - ‘’скелет’’ сайту,на якому визначенно розташування всіх елементів

CSS - ‘обкладинка’’ сайту, за допомогою нього ми визначаємо який буде дизайн сайту

JavaScript - мова програмування,яка була створенна щоб ‘оживити’ веб-сторінки
